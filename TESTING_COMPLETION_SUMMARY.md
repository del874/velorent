## 📝 测试最佳实践

### 编写好的测试
1. ** Arrange-Act-Assert 模式**
   ```typescript
   // Arrange
   const mockUser = { name: 'Test' }

   // Act
   render(<Component user={mockUser} />)

   // Assert
   expect(screen.getByText('Test')).toBeInTheDocument()
   ```

2. **描述性测试名称**
   - ✅ "should render user name when logged in"
   - ❌ "test user"

3. **测试独立性**
   - 每个测试应该独立运行
   - 不依赖其他测试的状态

4. **Mock 外部依赖**
   - 隔离数据库、API、文件系统
   - 使用 jest.mock()

### 测试数据管理
1. **使用测试数据库**
   - 独立的测试数据库文件

2. **测试数据工厂**
